## Overview

Allow provides:

- **Multiple Authentication Strategies** - Local, OAuth (GitHub, Google, Discord), JWT, custom
- **Account Linking** - Link multiple auth methods to one user
- **Session Management** - Secure session handling with database storage
- **Middleware Integration** - Seamless Verb framework integration
